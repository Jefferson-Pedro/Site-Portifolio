Introdução

Muito Prazer, Meu nome é Jefferson Pedro, tenho 30 anos, sou recém formado no curso Análise e Desenvolvimento de Sistemas pela Universidade Estácio de Sá.


Sou um profissional de TI com uma trajetória diversificada. Minha experiência inclui suporte técnico, consultoria e desenvolvimento. Hoje atuo no suporte a usuários externos, resolvendo problemas de hardware e software, gerenciando redes e compartilhando recursos. Além disso, tenho habilidades em análise de dados, segurança e criação de sistemas automatizados no oracle pbcs. Meu conhecimento abrange várias tecnologias, incluindo JAVA, PYTHON, TYPESCRIPT, bancos de dados MYSQL/POSTGREES e desenvolvimento web. Atualmente, estou conduzindo o Projeto NutriNotes, uma iniciativa solo em que aplico minhas habilidades de programação com JAVA (backend), ANGULAR (frontend) e MYSQL (banco de dados relacional) para criar uma solução personalizada para nutricionistas, otimizando seus processos nos atendimentos comerciais/industriais.
